# Network Programming Practices (BCA 6th Semester)

This repository contains Java-based network programming lab exercises and practices developed as part of the BCA 6th Semester curriculum.

## 📌 Overview
The projects focus on the fundamentals of networking using Java, covering TCP/UDP communication, URL processing, proxy authentication, and network security filters.

## 📂 Lab Exercises & Questions

| File / Folder | Description |
|:--- |:--- |
| **NISTTimeClient.java** | A client that connects to NIST servers to retrieve network time. |
| **SpamFilter.java** | Logic to identify and filter out unauthorized or spam addresses. |
| **ProxyAuthentication.java** | Implementation of authentication headers for proxy server access. |
| **URLParts.java** | Program to decompose a URL into Protocol, Host, Port, and Path. |
| **WebPageReader.java** | Basic implementation of a web crawler/page content downloader. |
| **Question6 / 7 / 8** | Solutions for specific lab manual requirements and socket logic. |

## 🛠️ Concepts Covered
* **Socket Programming:** TCP (ServerSocket/Socket) and UDP (Datagram).
* **InetAddress:** Working with IP addresses and hostnames.
* **URL & URI:** Parsing and connecting to web resources.
* **HTTP Protocols:** Managing headers and proxy settings.
* **Multithreading:** Handling multiple client connections (where applicable).
